# TDS Game

Welcome to my TDS game!

## How to Play

1. Download **full.zip**.
2. Extract the contents of the ZIP file.
3. Open **game.exe**.
4. In the console, enter a number to choose the **game speed**.
5. The game will start automatically.

## After the Game

If you lose the game, the console will ask if you want to play again.

* Type **y** to play again.
* Type **n** to exit the game.

## Notes

Make sure all files from **full.zip** stay in the same folder as **game.exe**, otherwise the game may not run correctly.
